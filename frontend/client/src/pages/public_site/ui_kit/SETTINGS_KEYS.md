# UI Kit — Settings Keys (settings_scoped)

مفاتيح التحكم المرن من لوحة الإعدادات لكل Website/Branch:

- public.ui.theme_json (object): ألوان/خطوط/Radius/Custom CSS
- public.ui.template_default (string): template pack code
- public.ui.custom_css (string)

نموذج theme_json:
{
  "mode":"auto",
  "colors":{"primary":"#0ea5e9","secondary":"#111827","accent":"#22c55e","bg":"#fff","fg":"#0b1220","muted":"#6b7280"},
  "fonts":{"base":"system-ui, ...","heading":"system-ui, ..."},
  "radius":"14px",
  "custom_css":""
}
